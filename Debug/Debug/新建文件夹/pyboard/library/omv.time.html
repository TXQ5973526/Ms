time — 跟踪运行时间¶

time 模块用于追踪运行时间。 import utime 用于标准python时间模块。

函数¶

time.ticks()¶

返回以毫秒计的通电后的运行时间。

注解

使用 pyb.millis() 而非该函数。

time.sleep(ms)¶

休眠 ms 毫秒数。

注解

使用 pyb.delay() 而非该函数。

构造函数¶

class time.clock¶

返回一个时钟对象。

Methods¶

clock.tick()¶

开始追踪运行时间。

clock.fps()¶

停止追踪运行时间，并返回当前FPS（每秒帧数）。

在调用该函数前始终首先调用 tick 。

clock.avg()¶

停止追踪运行时间，并返回以毫秒计的当前平均运行时间。

在调用该函数前始终首先调用 tick 。

clock.reset()¶

重置时钟对象。
